# odin-etch-a-sketch

A browser-based sketchpad application that lets users draw on a grid by hovering the mouse. This project is built as part of The Odin Project's Foundations curriculum to practice DOM manipulation, event listeners, and CSS Flexbox layout principles.

## Features

- **Dynamic Grid Generation:** Generates a custom grid layout programmatically using JavaScript instead of static HTML elements.
- **Flexbox Layout:** Utilizes CSS Flexbox to align grid squares dynamically inside a fixed-size container.
- **Interactive Sketchpad:** Simulates a drawing pen effect when the user moves their cursor over the grid squares.
- **Grid Resizing:** Includes a control button that prompts the user for a custom grid size (up to 100x100) and regenerates the workspace within the same physical dimensions.
- **Responsive Sizing:** Automatically calculates individual square dimensions based on the user's input so that the grid container's overall size remains constant.
